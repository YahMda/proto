Analisis Kelayakan Air untuk Pertanian — Machine Learning

Juli 2025

Ringkasan

Menganalisis dataset kualitas air untuk menilai kelayakan air dalam sektor agrikultur. Melakukan preprocessing, normalisasi, dan pembagian data latih/uji. Menerapkan beberapa model seperti KNN, Decision Tree, Random Forest, dan Logistic Regression; serta mengevaluasi menggunakan F1-Score. Model Random Forest menunjukkan performa terbaik.

Teknologi dan Tools

  • Python (scikit-learn, pandas, numpy)
  • Google Colab (link: Colab)
  • Data preprocessing, feature engineering, evaluasi model

Hasil & Temuan

Random Forest memberikan hasil terbaik berdasarkan metrik F1-Score pada dataset yang digunakan.
