Analisis Kelayakan Air untuk Pertanian Menggunakan Machine Learning

Tanggal: Juli 2025

Deskripsi:

Proyek ini bertujuan untuk menentukan kelayakan air dalam sektor pertanian dengan menganalisis dataset kualitas air. Data diproses melalui tahapan pembersihan, normalisasi, dan pembagian data latih & uji.

Model yang digunakan: KNN, Decision Tree, Random Forest, Logistic Regression.

Hasil: Model Random Forest memberikan akurasi dan F1-Score tertinggi.

Python Machine Learning Data Analysis

Link proyek: Google Colab
