IoT Palang Pintu Otomatis untuk Toko

Mei 2025

Deskripsi

Sistem IoT berbasis ESP32 untuk mengontrol palang pintu toko secara otomatis. Sensor ultrasonik mendeteksi kendaraan masuk, dan servo motor membuka palang. Data dikirim ke web dashboard untuk monitoring jarak dan status pintu secara real-time.

Komponen

  • ESP32
  • Sensor Ultrasonik HC-SR04
  • Servo Motor
  • WiFi Module (built-in)
  • Web Dashboard HTML, CSS, JS

Fitur

  • Palang otomatis ketika kendaraan terdeteksi
  • Kontrol manual via website
  • Monitoring real-time melalui WiFi
